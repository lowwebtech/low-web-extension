Embed video

Youtube

If video_clicktoload is enabled, Youtube iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Youtube). A click launch the video, low quality is not automatically selected.

note : quality setting is only available on Youtube.com (via an hack)

Vimeo

If video_clicktoload is enabled, Vimeo iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Vimeo). One (sometime two) click should launch the video.

note 1 : quality setting is only available for embed url with quality parameter

note 2 : presets Light = 240p, Normal = 360p

Dailymotion

If video_clicktoload is enabled, Dailymotion iframe should be blocked and replaced by an optimized iframe containing only an image, a title, a button and a link, no script (styled like original Dailymotion). A click should launch the video in a new tab without quality optimisation :(

note : quality parameters doesn't work
